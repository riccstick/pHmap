# pHmap
Software for generating a surface charge based pH-profile map.

## Requirements
- [ABPS](http://www.poissonboltzmann.org/) Adaptive Poisson-Boltzmann Solver
- [PDB2PQR](http://www.poissonboltzmann.org/)
